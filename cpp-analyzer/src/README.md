# src Directory (cpp-analyzer)

This directory contains all Java source code for the analyzers, GUI, networking, and utilities.

- `analyzer/` — Analyzer logic for C++, CUDA, Python
- `gui/` — JavaFX GUI code
- `network/` — Networking code for client/server
- `reports/` — (Reserved for report-related code)
- `resources/` — JavaFX and other resources
- `test/` — GUI and analyzer test code
- `utils/` — Utility classes

See each subdirectory for more details.
